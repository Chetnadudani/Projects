\# Retail Sales Analysis Using Python



\## Overview

This project analyzes retail sales data to understand revenue trends, product performance, and city-wise sales using Python.



\## Tools Used

\- Python

\- Pandas

\- Matplotlib

\- Jupyter Notebook



\## Key Analysis

\- Data cleaning and preprocessing

\- Monthly sales trend analysis

\- Product-wise revenue analysis

\- City-wise sales comparison



\## Insights

\- MacBook Pro Laptop is the highest revenue generating product.

\- Sales increase significantly during year end.

\- Major cities contribute majority of revenue.

\- Electronics dominate total sales.
